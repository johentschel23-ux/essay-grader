Error Loading Application

The PDF Viewer application could not load properly. This is likely due to an issue with locating the build files.

Troubleshooting steps:

  1. Make sure the application was built correctly with npm run build before packaging
  2. Check that the build directory is included in the packaged application
  3. Verify the path to index.html in the main.js file
  4. Try reinstalling the application

If the issue persists, please contact support or check the application logs for more details.
